## üìñ Description

Planora is a comprehensive project management platform designed to enhance team productivity and collaboration. Built with modern web technologies, it provides an intuitive interface for managing projects, tracking issues, and organizing sprints efficiently.

The system features role-based access control with distinct admin and user capabilities, enabling secure team management and project oversight. Users can create and manage multiple projects, track issues with detailed workflows, organize tasks into sprints, and collaborate through comments and real-time updates. The platform includes a personalized dashboard for monitoring project progress, favorite projects for quick access, and a responsive design that works seamlessly across all devices.

With secure authentication using JWT, email verification via Nodemailer, and a clean, dark-mode-enabled interface powered by Tailwind CSS, Planora delivers a professional project management experience suitable for teams of any size.

## üõ† Tech Stack

### Frontend
-   **Core**: React 19, Vite
-   **Routing**: React Router DOM
-   **Styling**: Tailwind CSS, Lucide React
-   **State Management**: Zustand
-   **Form Handling**: React Hook Form, Zod
-   **HTTP Client**: Axios
-   **UI Components**: React Toastify

### Backend
-   **Runtime**: Node.js
-   **Framework**: Express.js
-   **Database**: MongoDB (Mongoose)
-   **Authentication**: JWT (jsonwebtoken), bcrypt
-   **Email Service**: Nodemailer
-   **Utilities**: CORS, dotenv
